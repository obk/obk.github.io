## about
this repository contains the files for my personal website. the following files are included in site folder:

*   `favicon.ico`: icon file.
*   `public_key.asc`: my public key.
*   `cv.pdf`: my curriculum vitae in pdf format.
*   `bb.css`: css file that contains the styling for the website.
*   `index.html`: main html file for the website.
